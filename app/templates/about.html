Funcionamiento

Tenemos varios recursos que utilizaremos:

Funcionalidad básica:

  • Recurso: "/": Muestra una lista de los 5 aparcamientos con mas comentarios y,a la derecha, las páginas de usuario disponibles.
  • Recurso: "/aparcamientos": Muestra el listado de todos los aparcamientos de la C.M y un formulario de filtrado por distritos.
  • Recurso: "/aparcamiento/id": Siendo el id el id del aparcamiento en cuestion, muestra los datos almacenados del aparcamiento(extraido del XML).
  • Recurso: "/usuario": Siendo "usuario" un usuario registrado, muestra la página de ese usuario, con sus aparcamientos seleccionados.
  • Recurso: "/usuario/xml": Siendo "usuario" un usuario registrado, muestra el canal XML de dicho usuario.
  • Recurso: "/about": Muestra esta página.

Funcionalidad optativa:

  • Recurso: "/register": Permite dar de alta a un nuevo usuario.
  • Cada recurso "/aparcamientos/id" contiene un mapa que muestra la ubicacion del aparcamiento.
  • Uso de script en JavaScript sencillo para generar el mapa con el marcador.
  • Diseño de página web personal, junto con banner.
  • Uso de favicon.
  • Uso de CSS3 junto con HTML5.(el formulario del log-in no permite dejar un campo sin rellenar saltando un "pop-up")
  • CSS dinamico y controlado, solo cambia el estilo de la columa de la izuqierda, que es donde se muestran los contenidos
